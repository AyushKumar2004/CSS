Units

CSS has several different units for expressing a length,
Many CSS properties take a "length" values,such as width,margin,padding,font-size,etc.
Length is a number followed by a length unit,such as 10px,2em etc. 

            h1{
                font-size:60px;
            }

            p{
                font-size:25%;
                line-height:50em;
            }
    There are two types of length units:absolute and relative.
  1. Absolute
    • cm
    • mm
    • in
    • px :- 1px=1/96 of in
    • pt :- 1pt=1/72 of in
    • pc :- 1pc=12pt
  2. relative Length
    • em
    • rem
    • vw
    • vh
    • %

5.vmin:-relative to 1% of viewport smaller dimension means whoever is smallest height or width it will take the reference of smaller one

6.%:-relative to the parent element
