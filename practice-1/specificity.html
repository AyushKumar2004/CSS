CSS Specificity

If there are two or more css rules that point to the same element,the selector with the highest speifity value will "win",and its style declaration will be applied to that html element.

Inline css has higher priority followed by Internal and then External

Rank below high to low

  • inline CSS
  • ID Selector
  • Class Selector
  • tag name Selector
